Six takes on the
active pill.

The current brand-emerald active state reads as a primary action and competes with the Save & Generate CTA. Six candidate treatments — from semantic per-pill colors to subtle underlines — for the Bucket and Applies pills inside the recoup row.

VAR A

Brand emerald solid current

bg-brand-700

Solid emerald fill with white text — same vocabulary as the Save & Generate button. Trade-off: reads as primary action; competes for attention with the rail’s main CTA. Three of these in a single recoup row can feel loud.

RECOUP · 01
VAR B

Soft brand tint

bg-brand-50 + brand-700 border

Pale emerald fill with a clear emerald border and brand-800 text. Trade-off: quieter than the solid; dot keeps its own color cleanly without ringing. Reads as "selected" without claiming primary-action status.

RECOUP · 01
VAR C

Semantic per-pill

Bucket → its own dot color

Each bucket pill takes on its own semantic color when active — Inside → sky, Outside → amber, Off-top → rose. The dot color escalates to the full fill, reinforcing meaning. Applies pills (no semantic mapping) use ink-900 as a neutral.

VAR D

White + brand border + check

bg-white + ring-brand + ✓

Stays white; the affordance is a brand-emerald check mark and a thicker emerald border. Trade-off: very quiet — closest to a checkbox in feel. May not register as "selected" at a glance for users who are scanning quickly.

RECOUP · 01
VAR E

Brand-100 pale chip

bg-brand-100 + brand-300 border

Slightly more saturated than variation B — a pale chip rather than a hint of tint. Border is brand-300 (mid emerald) so the chip reads warm rather than outlined. Closest neighbor to the existing "Signed" badge color so the form vocabulary stays consistent.

RECOUP · 01
VAR F

Underline accent

bg-ink-50 + brand-700 underline

Faint ink-50 background with a 2px brand underline like a tab. Reads as "selected" via tab vocabulary rather than chip fill. Trade-off: works great horizontally, can look truncated when pills wrap to a new line.

RECOUP · 01
VAR G

Sky-700 solid

bg-sky-700 + white text

Same shape as the current brand variant, but in sky-700 blue. Trade-off: doesn’t compete with brand emerald (which is reserved for primary action), but introduces a new accent. Reads as "informational selection" rather than "primary action".

All seven, same pill

G · Sky-700 solid
My pick
C · Semantic per-pill. The bucket dot already encodes meaning (sky = inside, amber = outside, rose = off-top). Letting that color escalate to the entire pill when active turns the dot from decoration into the affordance — "this pill means inside-cap, and it's the one you picked". For the Applies row (no semantic colors), fall back to a quieter neutral so it doesn’t compete with the bucket choice.

Runner-up: B · Soft brand tint. If the multi-color treatment feels chaotic in practice, B’s pale fill + clear emerald border preserves "selected" without competing with the rail’s primary CTA.

Variation A (current) is solid but loud. Variations D and F are too quiet for a required field. E is OK but blends into the deal-type badges and signed-badge family. G works but introduces a second strong accent into a form that doesn’t need it.
